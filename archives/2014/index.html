新手使用Hexo搭建博客(Windows篇)

这是新的开始，

在这里，你可以拥有你自己的博客

首先了解下Hexo

Hexo是基于node.js的静态博客网站生成器，支持Markdown文件格式

准备工作

  • 安装 Node.js
  • 安装 Git
  • 注册 github账户

开始搭建Hexo

安装完Node.js后，通过node.js安装Hexo，也可以在dos下安装(默认在dos下)



安装完成之后，输入 Hexo - -version

确认安装，得到Hexo的版本的信息： hexo: 2.8.1

上传Hexo到你的github上

首先你需要有一个github账户，登录后点击右上方的 + 号 选择new repository。

在Repository name填上你的github用户名+github.io

例如用户名是test，则填上test.github.io。然后点击Create repository
